TEORÍA 01 : Introducción al CSS, conceptos básicos.

EXPLICACIÓN:

El lenguaje CSS(1) es un lenguaje de mediante el cual podremos dar estilo y organizar visualmente los documentos HTML.

Al igual que en HTML deciamos que es un lenguaje de etiquetas, CSS es un lenguaje de reglas.

Las reglas CSS las forman selectores, propiedades y valores:


	   h1    {   color   : black ; }
	           ---------   -----
	         { propiedad : valor ; }
	--------  ---------------------
	selector       declaración
	--------------------------------
	           Regla CSS

  • El selector : Indica el o los elementos a los que será aplicada la declaración.

  • La declaración : Hace referencia al cambio a aplicar a los selectores, está formada por reglas de estilo separadas por el simbolo punto y coma (;), estas reglas de estilo estan formadas por propiedades y valores.

    • Las propiedades : Son indicaciones preestablecidas que indican el tipo de cambio que se aplicará, pueden servir para cambiar la duente de un elemento (font-family), el color de un texto (color), el color de fondo (background-color), etc.

    • Los valores : Son los que definiran que estilo tiene el elemento, color rojo (red), fuente Arial (arial), etc.

Las reglas CSS se pueden incrustar en un documento HTML o indicar un vinculo a un documento .css que contenga dichas reglas.

  • Hay dos maneras de incrustar reglas CSS en un documento HTML, usando el atributo style="" donde entre las comillas insertaremos las declaraciones y también podemos insertarlo en la cabecera entre las etiquetas <STYLE> </STYLE>

  • Podemos vincular un archivo CSS donde iremos indicando las modificaciones de estilo mediante la inclusión en la cabecera de un vinculo a este archivo.

    • 
	<link rel="stylesheet" type="text/css" href="RUTA DONDE SE ENCUENTRA EL ARCHIVO CSS"/>
