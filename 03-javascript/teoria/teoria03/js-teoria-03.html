TEORÍA 03 : Estructuras condicionales simples.

EXPLICACIÓN:

Las estructuras condicionales hacen lo que su nombre indica.

Hay varias condicionales y son: IF

  • IF: La estructura condicional IF, comprobara una condición, si se cumple devolvera una respuesta y si no se cumple devolverá otra.

    Su sintaxis es:

    if ( aqui va la condición )
{ aquí va la resolucion de si se cumple }
else
{ aquí va la resolucion de si no se cumple }

    La condición esta compuesta por dos valores separados por un operador de comparación que son los siguientes:

    > mayor que
< menor que
>= mayor o igual que
<= menor o igual que
== igual a 
!= distinto de

    • Operadores Lógicos:

      Si queremos comprobar mas de una condicion tenemos dos operadores logicos Y ; O

      • Operador "Y": Operador "Y": Con el operador "Y" (en JAVASCRIPT &&) comprobaremos varias opciones y si todas ellas se cumplen, devolvera la condición verdadera.
      • Operador "O": Con el operador "O" (en JAVASCRIPT ||) se comprueban varias condiciones y si cualquiera de ellas es verdad devuelve la condición verdadera, solo si todas ellas son falsas nos devolverá la condición falsa.

    El shorthand para los condicionales IF es el siguiente:

    condicion ? valor si es verdad : valor si es falsa.
