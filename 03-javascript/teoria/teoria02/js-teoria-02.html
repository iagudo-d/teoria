TEORÍA 02 : Cuadros de dialogo en JAVASCRIPT.

EXPLICACIÓN:

Hay varios tipos de cuadros de dialogo: ALERT, PROMPT.

  • ALERT: Nos permitira abrir una alerta para el usuario.

    Su sintaxis es:

    alert("hola buenos días");

    Esto nos mostrará una alerta diciendo buenos dias

  • PROMPT: Sirve para que el usuario nos proporcione información.

    Su sintaxis es:

    prompt("¿Hola como te llamas?");

    Esto abrirá un cuadro de dialogo preguntando como te llamas con un formulario para introducir el dato solicitado.

    Se le puede añadir una segunda parte a modo de ayuda para rellenar el campo poniendo una coma tras la pregunta y el texto de ayuda, esto aparecerá dentro del campo del formulario.

    prompt("¿Hola como te llamas?,Escribe aqui tu nombre.");
En los cuadros de dialogo no se pueden usar etiqutas HTML, usaremos secuencias de escape:
  • \n : Para insertar un salto de linea como con <BR/>.
  • \t : Para tabular un texto.
  • \" : Para insertar comillas dentro de un cuadro de dialogo.
