TEORÍA 01 : Introducción a JAVASCRIPT, conceptos básicos.

EXPLICACIÓN:

JAVASCRIPT es un lenguaje de script que nos permite dinamizar las páginas web.

JAVASCRIPT es un lenguaje "case sensitive", es decir diferencia mayusculas y minusculas.

Todas las instrucciones se terminan mediante punto y coma (;).

Comentarios en JAVASCRIPT:

  • Para comentarios de una sola linea usaremos dos //.
  • Para comentarios multilinea usaremos /* ... */.

Se puede introducir de varias formas:

  • Dentro del documento HTML.
  • En un archivo vinculado.

Tipos de datos en JAVASCRIPT:

En este lenguaje no se define el tipo de datos al crear la variable, por defecto los datos serán considerados como texto, pero pueden ser de cuatro tipos:

  • Texto (String).
  • Numericos.
  • Booleanos (Verdadero o Falso).
  • Nulos (Null o sin valor).

TEORÍA 02 : Cuadros de dialogo en JAVASCRIPT.

EXPLICACIÓN:

Hay varios tipos de cuadros de dialogo: ALERT, PROMPT.

  • ALERT: Nos permitira abrir una alerta para el usuario.

    Su sintaxis es:

    alert("hola buenos días");

    Esto nos mostrará una alerta diciendo buenos dias

  • PROMPT: Sirve para que el usuario nos proporcione información.

    Su sintaxis es:

    prompt("¿Hola como te llamas?");

    Esto abrirá un cuadro de dialogo preguntando como te llamas con un formulario para introducir el dato solicitado.

    Se le puede añadir una segunda parte a modo de ayuda para rellenar el campo poniendo una coma tras la pregunta y el texto de ayuda, esto aparecerá dentro del campo del formulario.

    prompt("¿Hola como te llamas?,Escribe aqui tu nombre.");
En los cuadros de dialogo no se pueden usar etiqutas HTML, usaremos secuencias de escape:
  • \n : Para insertar un salto de linea como con <BR/>.
  • \t : Para tabular un texto.
  • \" : Para insertar comillas dentro de un cuadro de dialogo.

TEORÍA 03 : Estructuras condicionales simples.

EXPLICACIÓN:

Las estructuras condicionales hacen lo que su nombre indica.

Hay varias condicionales y son: IF, SWTCH.

  • IF: La estructura condicional IF, comprobara una condición, si se cumple devolvera una respuesta y si no se cumple devolverá otra.

    Su sintaxis es:

    if ( aqui va la condición )
{ aquí va la resolucion de si se cumple }
else
{ aquí va la resolucion de si no se cumple }

    El shorthand para los condicionales IF es el siguiente:

    condicion ? valor si es verdad : valor si es falsa.

    La condición esta compuesta por dos valores separados por un operador de comparación que son los siguientes:

    > mayor que
< menor que
>= mayor o igual que
<= menor o igual que
== igual a 
!= distinto de

    • Operadores Lógicos:

      Si queremos comprobar mas de una condicion tenemos dos operadores logicos Y ; O

      • Operador "Y": Operador "Y": Con el operador "Y" (en JAVASCRIPT &&) comprobaremos varias opciones y si todas ellas se cumplen, devolvera la condición verdadera.
      • Operador "O": Con el operador "O" (en JAVASCRIPT ||) se comprueban varias condiciones y si cualquiera de ellas es verdad devuelve la condición verdadera, solo si todas ellas son falsas nos devolverá la condición falsa.

  • SWITCH: Es una alternativa al IF pero solo se puede aplicar cuando la condicion es de igualdad

    Su sintaxis es:

    Escribiremos SWITCH y entre parentesis la variable que queremos contrastar.

    Abriremos llaves y escribiremos los CASE seguidos de el valor al que queremos igualar, despues exscribiremos los

    Al final del switch y antes de cerrar las llaves, con la instruccion default seguida de dos puntos(:) podremos especificar las instrucciones si no se cumple ningun "case"

    Podemos poner tantos CASE como queramos y será como si pusieramos el condicional "O", no se pueden contrastar varias condiciones como cuando usamos la funcion "Y" puesto que estas son excluyentes.

    
switch ( aqui va la variable que vamos a analizar )
{ "case" el valor al que la igualamos : las instrucciones si se cumple la igualdad;
break; }

TEORÍA 04 : Los bucles.

EXPLICACIÓN:

Las bucles sirven para ejecutar una accion varias veces.

Hay varias tipos de bucles: WHILE, FOR

  • WHILE: Permite ejecutar una instruccion varias veces mientras se cumpla la condicion.

    Su sintaxis es:

    WHILE y entre parentesis la condicion, tras esto y entre llaves escribiremos las instrucciones que ejecutará si se cumple la la condición.

    while ( aqui va la condicion )
{ Las instrucciones que se ejecutarán mientras la condicion sea verdad }

    var i=1;			<---INICIALIZAMOS LA VARIABLE

while ( i < 10 )	<--MIENTRAS SE CUMPLA i MENOR QUE 10
{alert( i ) }		<--MUESTRA UN ALERT CON EL VALOR DE i

  • FOR: Es una estrucutra repetitiva como WHILE pero que solo podremos utilizar cuando conocemos la cantidad de repeticiones.

    Su sintaxis es:

    FOR y entre parentesis iniciamos la variable un punto y coma (;) ponemos la condicion, otro punto y coma(;) y el incremento positivo o negativo. Tras esto entre llaves las instrucciones que se repetiran mientras se cumpla la condición.

    var i="";
for   ( inicializamos la variable   ;   condición   ;   incremento positivo o negativo )
{  instruciones que ejecuta mientras se cumpla la condición }

    var i="";
for (i=1 ; i<=10 ; i=i+1 )
{ Las instrucciones que se ejecutarán mientras la condicion sea verdad }

TEORÍA 05 : Funciones.

EXPLICACIÓN:

Una funcion es un conjunto de operaciones que resuelven un problema.

Podemos llamarlas en cualquier parte del programa.

Su sintaxis es:

function + nombre de la funcion + (argumentos separados de comas ",") + {codigo de la funcion y punto y coma ";"}

function nombre de la funcion ( argumento1 , argumento2 ... )
{ codigo de la funcion + ; }

Para llamar a una funcion usaremos su nombre, poniendo el valor de sus argumentos si los tiene.

Existen funciones con argumentos y sin argumentos.

  • Funciones sin argumentos:

  • Funciones con argumentos:

    Los parametros se crearan genericos en la funcion dentro de los parentesis y separados entre comas y se utilizarán en las instrucciones de la función dentro de las llaves.

    Llamaremos a la funcion indicando sus parametros, que pueden ser fijos o variables.

Por defecto las funciones no me permiten trabajar con su resultado. Podemos usar "return" para poder trabajar con un resultado de una función.(No se puede devolver varios resultados.)

Las variables que creamos en una función, solo tienen validez dentro de la función a no ser que las retornemos.

TEORÍA 08 : Funciones de texto en JAVASCRIPT.

EXPLICACIÓN:

Las fun

  • Función length:

    Nos devuelve el tamaño de un texto, el número de caracteres que tiene ese texto.

    Su sintaxis es:

    (donde buscará).length()

  • Función toLowerCase:

    Con esta función convertiremos una cadena de texto en minúsculas.

  • Función toUpperCase:

    Con esta función convertiremos una cadena de texto en mayúsculas.

  • Función indexOf:

    Con esta función buscaremos en un texto una cadena que buscamos, en caso afirmativo nos devolverá la posición mediante un número empezando por 0 seguido de su posición y en caso negativo nos devolverá -1.

    Además esta función nos permite un segundo parametro que será la posicion donde queremos comenzar a buscar.

  • Función substring:

    Nos permite extraer un texto de el interior de otro, tiene dos argumentos o parametros, la posición desde donde comenzamos a extraer y el segundo es la posición final donde queremos parar de extraer, SIN incluir el ultimo caracter.

    Si solo ponemos un argumento, extraerá desde esa posición hasta el final del texto.

  • Función replace:

    Nos permite reemplazar un texto con otro.

    Su sintaxis es:

    La funcion y entre parentesis el texto a buscar y sustituir coma y el texto que sustituirá al anterior.

    Solo cambiará el primer texto que encuentre.

    .replace ("texto a cambiar","texto con el que lo sustituira")

  • Función substr:

    Nos permite extraer un texto que este dentro de otro texto, el primer argumento es la posición de inicio desde donde queremos empezar a extraer y el segundo argumento es el numero de caracteres que queremos que extraiga.

    Su sintaxis es:

    La funcion y entre parentesis el texto a buscar y sustituir coma y el texto que sustituirá al anterior.

    Solo cambiará el primer texto que encuentre.

    .substr (posicion de inicio, longitud de extracción)

TEORÍA 09 : Funciones numéricas en JAVASCRIPT.

EXPLICACIÓN:

Las fun

  • Función isNaN (is not a number):

    Con esta función comprobaremos si un valor es un número o no, si es cierto que es un número continuará.

    Su sintaxis es:

    isNaN(variable a comprobar)

  • Objeto MATH:

    Nos permitirá trabajar con funciones matemáticas para calcular el máximo el mínimo, exponentes, raices, números aleatorios, etc.

    • Math.max: con esta función calcularemos el nuumero mayor de una serie de números.

      Su sintaxis es:

      Math.max(var 1,var 2,....var x)

    • Math.min: con esta función calcularemos el nuumero menor de una serie de números.

      Su sintaxis es:

      Math.min(var 1,var 2,....var x)

    • Math.ads: Con esta función calcularemos valor absoluto de un número, solo admite una variable o numero

      Su sintaxis es:

      Math.ads(var o numero)

    • Math.pow: con esta función calcularemos el exponente de un número, tiene dos elementos el primero el numero elevado y el segundo el exponente.

      Su sintaxis es:

      Math.pow(num1,exponente)

    • Math.sqrt: con esta función calcularemos la raiz cuadrada de un número, solo admite una variable o numero.

      Su sintaxis es:

      Math.sqrt(variable o numero)

    • Math.random: nos devolverá un numero aleatorio entre 0 y 1.

      Su sintaxis es:

      Math.random()

    • Math.ceil: Sirve para redondear hacia en entero mayor.

      Su sintaxis es:

      Math.ceil(variable o numero)

    • Math.floor: Sirve para redondear hacia en entero menor.

      Su sintaxis es:

      Math.floor(variable o numero)

    • Math.round: Sirve para redondear hacia el entero más próximo.

      Su sintaxis es:

      Math.round(variable o numero)
