TEORÍA 11 : Etiqueta <PRE> de formato preformateado.

EXPLICACIÓN:

La etiqueta <PRE> es una etiqueta en bloque.
También se puede definir como etiqueta de estilo de texto o formato.
Se utiliza para mantener el formato del texto igual que lo escribimos en codigo y utiliza una fuente similar a la de la etiqueta <TT>.
La etiqueta <PRE> es la encargada de mantener el estilo y formato del texto.

EJEMPLO:

Un ejemplo de como veriamos el efecto de esta etiqueta es el que usamos en la unidad dos para mostrar la estructura basica de una web: 

						
	 <html> 
		 <head> 
			 <title>  TITULO  </title> 
		 </head> 
		 <body> 
		 </body> 
	 </html>

PROPIEDADES:

La etiqueta <PRE> no tiene ninguna propiedad que la modifique.
