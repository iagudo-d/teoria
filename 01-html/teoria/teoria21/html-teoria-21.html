TEORÍA 21 : Etiqueta <CODE> inserción de textos con formato código.

EXPLICACIÓN:

La etiqueta <CODE> es una etiqueta en linea.

Su cometido es ornamental.

Esta etiqueta la usaremos cuando queramos definir que el texto al que hace referencia es un texto de código computacional, y le dotará de un aspecto visual similar al código que se usa abitualmente en programación y computación.

EJEMPLO:

Ejemplo de <CODE>:

Sintaxis básica:

<CODE> Texto con formato de código </CODE>

Esto es lo que hace el código anterior, en naranja el texto con formato de código:

Asi quedaría un texto con formato de código.

PROPIEDADES:

La etiqueta <CODE> no admite propiedades específicas.
