TEORÍA 12 : Etiqueta <BR>, insertar salto de linea.

EXPLICACIÓN:

La etiqueta <PRE> es una etiqueta en bloque.
Es una etiqueta estructural.
Se utiliza para insertar un salto de linea.

EJEMPLO:

Para explixar el efecto de esta etiqueta vamos a ver un ejemplo sin usarla y otro usando la etiqueta, en el primer ejemplo vamos a ver varias palabras separadas por un Intro en HTML y veremos como no reflejan el salto de linea, en el segundo ejemplo vamos a ver las mismas palabras separadas unicamente por la etiqueta <BR> y podremos apreciar como despues de cada <BR> pasará a la siguiente linea.

  • Ejemplo de código sin la etiqueta <BR>: 

    • 				
	Coche.
	Casa.
	Árbol.

  • Así se muestra en el navegador:

    • Coche. Casa. Árbol.

  • Ejemplo de código con la etiqueta <BR>: 

    • 					
	Coche.<BR/>Casa.<BR/>Árbol.<BR/>

  • Así se muestra en el navegador:

    • Coche.
    Casa.
    Árbol.

PROPIEDADES:

La etiqueta <BR> tiene una propiedad que es "clear".

  • clear: Establece si un objeto permite objetos flotantes a sus lados o no, sus posibles valores son: none, left, right y both.
