TEORÍA 10 : Etiqueta <FONT> para edicion de fuente.

EXPLICACIÓN:

La etiqueta <FONT> es una etiqueta en linea.
También se puede definir como etiqueta de estilo de texto.
Su único cometido es el de ornamentar el texto.
La etiqueta <FONT>, es la encargada de dar al texto estilo, color y tamaño mediante la inclusión en ella de varias propiedades ya que sin ellas esta etiqueta no hace nada, al menos tiene que tener una de ellas con un valor.

EJEMPLO:

Para poder ver el aspecto visual de esta etiqueta pondremos un texto de referencia sin usar la etiqueta <FONT> otro usandola pero sin incluir propiedades y otra igual en color rojo, fuente verdana y tamaño 3 mediante la inclusión de propiedades en la etiqueta <FONT>.

  • Este es el formato de fuente si no usamos la etiqueta:

    Ej.- Lorem ipsum dolor sit amet, consectetuer adipiscing elit. Maecenas porttitor congue massa. Fusce posuere, magna sed pulvinar ultricies, purus lectus malesuada libero, sit amet commodo magna eros quis urna. Nunc viverra imperdiet enim.


  • Este es el formato de fuente si usamos la etiqueta sin propiedades:

    Ej.- Lorem ipsum dolor sit amet, consectetuer adipiscing elit. Maecenas porttitor congue massa. Fusce posuere, magna sed pulvinar ultricies, purus lectus malesuada libero, sit amet commodo magna eros quis urna. Nunc viverra imperdiet enim.


  • Este es el formato de fuente si usamos la etiqueta aplicando propiedades y valores:

    Ej.- Lorem ipsum dolor sit amet, consectetuer adipiscing elit. Maecenas porttitor congue massa. Fusce posuere, magna sed pulvinar ultricies, purus lectus malesuada libero, sit amet commodo magna eros quis urna. Nunc viverra imperdiet enim.

PROPIEDADES:

La etiqueta <FONT> puede tener varias propiedades para ajustar diferentes parametros, son las siguientes: "color", "face" y "size".

  • color: Establece el color de la fuente, sus valores pueden ser nombres de colores en inglés o codigos de color hexadecimal precedidos del simbolo #.

    • Ej. color="red" o color="#ff0000" : Usando este valor el color de la fuente será Rojo.

      • Ej. Fuente de color rojo.

      Ej. Fuente de color azul.

      Ej. Fuente de color verde.

  • face: Establece el tipo de fuente que se utilizará.

    • Ej. face="verdana" : Usando este valor el tipo de fuente será Verdana.

      • Ej. Fuente arial.

      Ej. Fuente verdana.

      Ej. Fuente cambria.

  • size: Establece el tamaño de la fuente, sus posibles valores son los numeros del 1 al 7 .

    • Ej. size="1" : Usando este valor el tamaño de la fuente será el tamaño mas pequeño posible.

      • Ej. Tamaño de fuente 7.

      Ej. Tamaño de fuente 6.

      Ej. Tamaño de fuente 5.

      Ej. Tamaño de fuente 4.

      Ej. Tamaño de fuente 3.

      Ej. Tamaño de fuente 2.

      Ej. Tamaño de fuente 1.
