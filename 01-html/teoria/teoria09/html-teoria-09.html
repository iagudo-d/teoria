TEORÍA 09 : Etiqueta <TT> para fuente truetype.

EXPLICACIÓN:

La etiqueta <TT> es una etiqueta en linea.
También se puede definir como etiqueta de estilo de texto.
Su único cometido es el de ornamentar el texto.
La etiqueta <TT> de truetype, es la encargada de dar al texto el estilo de fuente TRUETYPE, este tipo de caracter ocupa un espacio horizontal homogeneo en todos los caracteres.
