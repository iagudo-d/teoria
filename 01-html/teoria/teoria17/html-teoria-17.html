TEORÍA 17 : Etiquetas <BIG> y <SMALL>, tamaño de texto.

EXPLICACIÓN:

Las etiquetas <BIG> y <SMALL> son etiquetas en linea.

Son etiquetas ornamentales.

Las utilizaremos para dar un formato de texto grande o pequeño respectivamente, sin usar atributos de tamaño de texto. Se pueden anidar una dentro de otra para conseguir aunmentar el tamaño de la letra mas de un punto.

EJEMPLO:

Ejemplo de <BIG>:

Sintaxis básica:

.Texto normal. <BIG> .Texto grande. </BIG> .Texto normal.

Esto es lo que hace el código anterior, en naranja el texto grande:

.Texto normal. .Texto grande. .Texto normal.

Ejemplo de <SMALL>:

Sintaxis básica:

.Texto normal. <SMALL> .Texto pequeño. </SMALL> .Texto normal.

Esto es lo que hace el código anterior, en naranja el texto pequeño:

.Texto normal. .Texto pequeño. .Texto normal.

PROPIEDADES:

Las etiquetas <BIG> y <SMALL> no admiten propiedades específicas.
