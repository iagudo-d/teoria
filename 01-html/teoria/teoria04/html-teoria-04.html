TEORÍA 04 : Etiqueta < P > creación de parrafos.

EXPLICACIÓN:

Con la etiqueta <P> podremos insertar parrafos dento de una página, de esta forma crearemos un elemento estructural que conformará un bloque, tras este elemento se dejará un doble espacio antes del siguiente elemento.
